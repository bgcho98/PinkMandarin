개인정보처리방침

Petoxia · 시행일: 2025년 3월 1일

PinkMandarin(이하 "회사")은 Petoxia(이하 "앱") 이용자의 개인정보를 중요시하며, 개인정보보호법 등 관련 법령을 준수합니다.

1. 수집하는 개인정보

수집 항목수집 방법
Google 계정 정보 (표시 이름, 이메일)Google 로그인 시 자동 수집
앱 사용 통계 (기기 내 앱별 사용 시간)사용 정보 접근 권한 허용 시
앱 내 활동 데이터 (코인, 목표, 펫 정보, 일일 통계)앱 이용 과정에서 자동 생성

2. 개인정보의 수집 및 이용 목적

  • 사용자 식별 및 로그인 기능 제공
  • 스마트폰 사용 목표 관리 및 달성 여부 확인
  • 일일 정산, 코인 지급, 리더보드 기능 제공
  • Firebase를 통한 클라우드 동기화

3. 개인정보의 보유 및 이용 기간

  • 로컬 데이터 (Room DB): 앱 삭제 시 즉시 삭제
  • 서버 데이터 (Firebase): 회원 탈퇴 시 삭제 또는 개발자에게 삭제 요청 가능

4. 개인정보의 제3자 제공

회사는 사용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, Google Firebase를 인증, 클라우드 기능 및 데이터 저장을 위해 사용하며, 이 경우 Google의 개인정보처리방침이 적용됩니다.

5. 사용 정보 접근 권한

본 앱은 스마트폰 사용 시간 측정을 위해 Android의 "사용 정보 접근" 권한을 요청합니다. 이 권한은 오직 사용 목표 달성 여부 확인을 위해서만 사용되며, 수집된 데이터는 기기에 보관되고 외부로 전송되지 않습니다. 사용자는 시스템 설정에서 언제든지 이 권한을 철회할 수 있습니다.

6. 아동의 개인정보 보호

본 앱은 만 13세 미만의 아동으로부터 의도적으로 개인정보를 수집하지 않습니다.

Privacy Policy

Petoxia · Effective: March 1, 2025

PinkMandarin ("the Company") values the personal information of Petoxia ("the App") users and complies with the Personal Information Protection Act and related laws.

1. Personal Information Collected

InformationCollection Method
Google account details (display name, email)Automatically collected upon Google sign-in
App usage statistics (per-app usage time on device)When Usage Access permission is granted
In-app activity data (coins, goals, pet info, daily statistics)Automatically generated during app usage

2. Purpose of Collection and Use

  • User identification and login functionality
  • Smartphone usage goal management and progress tracking
  • Daily settlement, coin distribution, and leaderboard features
  • Cloud synchronization via Firebase

3. Retention and Use Period

  • Local data (Room DB): Deleted immediately upon app uninstallation
  • Server data (Firebase): Deleted upon account deletion or by request to the developer

4. Provision to Third Parties

The Company does not provide users' personal information to third parties. However, Google Firebase is used for authentication, cloud functions, and data storage, in which case Google's Privacy Policy applies.

5. Usage Access Permission

This App requests Android's "Usage Access" permission to measure smartphone usage time. This permission is used solely to determine goal achievement. Collected data remains on the device and is not transmitted externally. Users may revoke this permission at any time in system settings.

6. Children's Privacy

This App does not intentionally collect personal information from children under 13 years of age.
